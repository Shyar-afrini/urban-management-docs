### Create BackBlaze account

Create a BackBlaze account, then create a new bucket

<Callout icon="key" color="#FFC107" iconType="regular">
  Retrieve all the necessary environment variables and include them in the Env
  file
</Callout>

```bash BackBlaze environment
B2_KEY_ID=<b2-application-id>
B2_APPLICATION_KEY=<b2-application-key>
BUCKET_NAME=<b2-bucket-name>
```
