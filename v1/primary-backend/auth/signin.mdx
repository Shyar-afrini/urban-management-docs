# POST /auth/signin

Sign in a user with email and password. Returns a session token you can use for authenticated requests.

### Request Body

<ParamField path="body" type="string" required>
  email
</ParamField>
<ParamField path="body" type="string" required>
  password
</ParamField>

```mermaid
sequenceDiagram
    participant Client
    participant Server

    Client->>Server: Send credentials (username & password)

    Server-->>Client: { "success": true }
    Note over Server,Client: Set-Cookie: session_token
```

### Helpers

Type and schema helpers for parsing and inferring types

```ts AuthInput
//? Use this for type infer
const authInputData: AuthInput = { email, password };
```

```ts AuthInputSchema
function parse(
  schema: ZodSchema<AuthInputSchema>,
  data: unknown
): TResult<AuthInput> {
  const result = schema.safeParse(data);
  if (!result.success) {
    return {
      success: false,
      error: {
        code: BAD_REQUEST_STATUS,
        message: BAD_REQUEST_ERR,
        details: result.error.flatten(),
      },
    };
  }
  return { success: true, data: result.data };
}
```
